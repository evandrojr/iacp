Features

🤖

AI-Generated Messages

Your git diff is analyzed by a local or remote LLM to produce clear, conventional commit messages.

🔌

Multi-Provider

Supports llama.cpp, Ollama, and AI-gatiator cloud gateway — switch freely between local and cloud models.

🎨

TUI Model Selector

Run iacp -s to browse and select AI models through a beautiful Bubble Tea terminal interface.

⚡

Fast Mode

Use iacp -f to skip the editor and commit instantly — ideal for quick, safe changes.

🛡️

Context-Aware

Displays model context limits and warns you before the diff exceeds them. Suggests switching models when needed.

🔍

Diff Preview

See your staged changes side-by-side with the generated message before committing.

Quick Start

Install

git clone https://github.com/evandrojr/iacp.git
cd iacp
go build -o iacp main.go
cp iacp ~/bin/

Usage

  1. Select your AI model: iacp -s
  2. Make some changes in your repo, then run iacp
  3. Review the AI-suggested message in the TUI editor, or use iacp -f for instant commits

Example

# Select a model (first time)
$ iacp -s

# Stage, generate, commit & push
$ iacp

Configuration

Your settings are stored in ~/.config/iacp/config.json. You can edit it directly or use iacp -s to switch models interactively.

{
  "provider":      "ollama",
  "model_path":    "llama3.2",
  "context_limit": 8192
}
